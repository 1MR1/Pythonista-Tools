FAQ

How Do I Request a Repo/Script?

Don't see a repo/script on here? Open a new issue with the name and url of the program you'd like added and we'll be sure to get on it ASAP. Also, please be sure to add the proper labels to your issue. It helps us and in return will get it on Pythonista Tools quicker.

How do I download scripts?

Pythonista Tools is currently just a collection of links to different projects made in Pythonista, so if there is something you want to download, go to its link and follow the instructions the project itself provides. If no instructions can be found at the project’s homepage, try asking about it on the Pythonista forums.
